##### 华为手机自带浏览器

- 问题：复制页面代码后发现click事件失效，后经过排查发现华为手机自带浏览器居然不支持ES6语法，必须要用babel转换成ES5才可以；

- 代码：

```javascript

var times = 60,timer;
$(".btn").on("click",function(){
    timer = setInterval(() =>{
        times --;
        $(".text").text(times + “s”);
        if(times < 0){
            $(this).text("tryagain");
            times = 60;
            clearInterval(timer);
        }
    })
})
// 注意这里 用了ES6的 箭头函数
```

- 处理：将计时器改成ES5的写法后，btn的click事件终于ok了；好坑！
